Information about Games as Drinks

Games as Drinks is a website that reimagines a range of different games as a selection of drinks, ranging from milkshakes to Sake. The website has been built under the guidance of CodeFirstGirls, a huge thanks to them! We hope you enjoy exploring Games as Drinks!

A huge thanks to Code First Girls for running this course!
